# 经典街机游戏克隆

这是使用JavaScript对象修饰模式实现的街机游戏"青蛙过河"，玩家可以通过上下左右键移动角色，直到到达河边即可通关。但是在移动过程中需要避开甲虫和障碍物，如果碰撞了甲虫角色就会被重置到起始位置。游戏中还设有多种不同效果的道具。
